### サーバーを立ち上げる方法
ローカルで実行する分にはhtmlファイルを直接ブラウザで開けば大丈夫です。
VM等を利用されている方は、以下の方法でサーバーを立ち上げることができます。

#### python
python -m SimpleHTTPServer

ポート番号を指定したいとき
例）8888
python -m SimpleHTTPServer 8888

#### nodejs
npm install
grunt server
